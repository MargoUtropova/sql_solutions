# SQL Solutions

В этой директории находятся решения задач на SQL.

## Навигация по задачам

Задачи разбиты по уровню сложности:

*   **[Easy](./easy/)** - базовые запросы, SELECT, JOIN, WHERE.
*   **[Medium](./medium/)** - задачи на оконные функции, подзапросы, CTE.
*   **[Hard](./hard/)** - сложные многоэтапные запросы.

## Формат решений

Каждый SQL-файл содержит:
1.  Краткое описание задачи.
2.  Схему таблиц (если нужно).
3.  Сам запрос, отформатированный для удобства чтения.
4.  Комментарии к ключевым моментам запроса.

## Примеры тем

*   Базовые запросы и фильтрация
*   Агрегатные функции (`GROUP BY`, `HAVING`)
*   Различные типы `JOIN`
*   Оконные функции (`ROW_NUMBER()`, `RANK()`, `LAG()`, `LEAD()`)
*   Работа с датами и временем
